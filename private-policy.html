M. Cookies

Our site uses cookies. A cookie is a file containing an identifier (a string of letters and numbers) that is sent by the web server to the web browser and stored by the browser. The ID is then sent back to the server every time the browser requests a page from the server. Cookies can be either "persistent" or "session": persistent cookies are stored by the web browser and remain valid until their specified expiration date, unless the user deletes them before the expiration date; session cookies, on the other hand, expire at the end of the user's session when the web browser is closed.
